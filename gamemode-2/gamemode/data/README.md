# data

**Ownership:** JSON/config data files — arenas, weapons, clothing, consumables.

**Structure:**
- `arenas/` — Arena preset JSON (currently `arenas.json` at root)
- `weapons/` — Weapon metadata, attachments, rotation
- `clothing/` — Clothing limits, presets
- `consumables/` — Consumable definitions

**Note:** `arenas.json` currently lives at `data/arenas.json`. Consider moving to `data/arenas/` in a later pass.
